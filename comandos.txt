git init: inicia um respositório local; 

git config: configura informações de usuário que estarão relacionadas aos commits

git init: inicia um repositório local e vazio referente ao diretório corrente;

git add: adiciona arquivos especificos ou todos os arquivos do diretório na branch atual do repositório local. 

git status: retorna quais arquivos estão em staging(estado temporário de memória);
